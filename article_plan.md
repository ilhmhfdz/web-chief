# Allow Adding Images to Articles and Editing Drafts

This plan details the implementation for allowing users to edit article drafts (whether generated by AI or manually) and inserting images with flexible positioning.

## Proposed Changes

1. **Install TipTap Editor Packages**
   - `@tiptap/react`, `@tiptap/pm`, `@tiptap/starter-kit`, `@tiptap/extension-image`, `@tiptap/extension-text-align` for rich text editing and image support with alignment.

2. **API: Image Upload Endpoint**
   - Create `app/api/admin/upload/route.ts` to handle multipart/form-data.
   - The endpoint will save the uploaded file to `public/uploads/` and return the public URL (`/uploads/filename.ext`).

3. **API: Update Article Endpoint**
   - Modify the existing `PATCH /api/admin/articles/route.ts` to support updating `title`, `slug`, `current_content`, `meta_description`, and `geo_keywords` in addition to `is_active`.

4. **UI: TipTap Editor Component**
   - Create `components/admin/RichTextEditor.tsx` that wraps TipTap.
   - Provide a toolbar with basic formatting (Bold, Italic, Headings) and Image tools (Upload Image, Align Left, Center, Right).
   - Implement custom logic for the image uploader to call the new upload API.

5. **UI: Edit Article Modal**
   - Update `ArticlesPageClient.tsx` to add an "Edit" button next to each article.
   - Create an `EditArticleModal` similar to `CreateArticleModal`, but pre-filled with the article's data.
   - Replace the generic `<textarea>` for HTML with the new `RichTextEditor` component.

## User Review Required

> [!IMPORTANT]
> The image uploads will be saved to the local `public/uploads/` directory. This works well for local development and traditional VPS deployments. However, if you are deploying to a serverless platform like Vercel, the local filesystem is read-only or ephemeral, meaning uploaded images will disappear after a deployment or server restart. If you plan to deploy to Vercel, we would need to integrate a cloud storage service like AWS S3 or Vercel Blob instead.
> **Is local storage acceptable for now?**

## Verification Plan

- Create a draft manually.
- Click "Edit" and use the rich text editor.
- Upload an image, change its alignment.
- Save the draft and ensure the updated content is persisted to the database.
- View the published article to confirm the image displays correctly.
